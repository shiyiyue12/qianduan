# 04

## 概述

CSS全称层叠样式表

网页的内容，由HTML控制

网页的样式，由CSS控制

## CSS语法

```html
h1
{
    color:red;
}

选择器{
    属性名:属性值;
    属性名:属性值;
}

```

## 常用的CSS选择器

元素选择器:h1,img,p 直接通过标签找到

类选择器:.className

id选择器:#id

通配符选择器:*

id选择器只能指定一个，class可以指定多个

一般不用id选择器，用class选择器

## 常用的CSS属性

字体大小:font-size

字体颜色:color

宽度:width

高度:height

背景色:background-color

文本水平居中:text-align

文本行高（垂直居中）:line-height

---

chrome浏览器：font-size最小值是12px

## 设置最高权重

!important
